🕹️ Easy and Quick Setup
Add content and deploy!

Get up and running within minutes. Just install the starter, add your content, deploy it, and there you go! This starter works seamlessly with hosts like Netlify. Photo by Clément H on Unsplash.

Content IntegrationReady-2-Deploy
📓 Markdown Content
No external CMS needed!

This starter uses MDX to integrate your content. MDX is basically a Markdown format that allows you to enrich your content with React components. This makes it fully customizable without external dependencies.

MarkdownFully Customizable
🧰 Extendable Layout
Add more sections!

The starter includes predefined sections as well as a template for new, custom sections. Moreover, you can add new projects to this section without additional coding. Photo by Kelly Sikkema on Unsplash.
